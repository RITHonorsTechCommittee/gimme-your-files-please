{{ appName }} - A simple file management tool.

Motivation

Google Drive is a fantastic tool, especially because Google accounts are so prevalent. However, sharing in Google Drive is done on a file-by-file basis, with no bulk permission editing tools. We found that as document collaborators came and went, there wasn't an efficient way to clean the old users out, especially if they owned the files. If you own a file in Google Drive (which happens if you create a file or folder), only you can transfer ownership to another user and no one can remove your editing permissions as long as you own the file. It would be unrealistic to ask users to go through their files one-by-one and transfer them to the people we want as new owners, so we built this tool.

What {{ appName }} Does

At it's most basic level, {{ appName }} allows you to see all of the users who have access to any file contained in a folder. After opening a folder with {{ appName }} from Google Drive, {{ appName }} digs through the folder and any subfolders, compiling a list of all the files it encounters and all the users who can access them. When it is done, it shows you a table of all the users it found, along with how many files they can access. Access is broken down into owner, writer, and reader and hovering over the quantities shows a preview of the files they own.

In addition to seeing the users who have access to files, {{ appName }} allows you to make bulk permission changes to the folder you opened. Clicking the transfer button, , sends an email to the person you selected asking them to transfer all their owned files to you. When they click the link in the email, they will be prompted to confirm the ownership transfer. Clicking the write button, , removes write access for the person on all their files. Clicking the read button, , removes read access for the person on all their files.

Currently, the transfer ownership button only works to transfer files between people in the same domain (the part after '@' in your email), but we are working to make it work for everyone.

FAQ

  • Why do I have to log in so many times?

    You are probably using multiple accounts. Due to the limitations of the tools provided to develop apps that work with Google Drive, we have to ask you which account to use often. If you find this limitation problematic, we suggest using a "private" or "incognito" window and logging in to only the account you wish to use with this app.
